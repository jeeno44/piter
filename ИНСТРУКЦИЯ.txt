Перед установкой внимательно ознакомьтесь с инструкцией по установке.

Установка проекта с github

1) Зайти в директорию в которой будет находиться проект

2) Выполнить команду git clone https://github.com/jeeno44/piter.git
!!! ВАЖНО !!!
У вас должен быть установлен git, иначе будет ошибка

3) Зайдите на ваш сервер mysql выполните SQL поочерёдно запросы из файла sql_queries.txt

4) Отредактируйте файл "append.php" с настройками вашего сервера mysql (строки с 6 по 10)

5) Зайдите в консоли в папку с проектом (cd <projects>/piter/) и выполните команду "php append.php"
   эта команда добавит в базу данных пользователей, посты и комментарии к постам

6) Запустите сервер в папке с проектом (<projects>/piter/) командой - "php -S 127.0.0.1:2020"

7) Откройте в браузере адрес 127.0.0.1:2020
Вам будет видна страница поиска постов по комментариям
